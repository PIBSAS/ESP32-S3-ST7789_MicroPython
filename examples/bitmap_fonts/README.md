# bitmap_fonts.py

## Required fonts:
  - [vga.py](/fonts/bitmap/vga.py)
  - [vga_8x16.py](/fonts/bitmap/vga_8x16.py)
  - [vga_bold_16.py](/fonts/bitmap/vga_bold_16.py)
  - [vga_bold_16x32.py](/fonts/bitmap/vga_bold_16x32.py)

Bitmap fonts use `text` method.
